## Note app
Users can create, update and delete notes.

### Instructions
1. Download the source code
``` bash
git clone 'https://github.com/10916/wt_cw2.git'
```
2. Install dependencies
``` bash
npm i 
```
3. Run the app
``` bash 
node app
```

#### Dependencies
- express 
- pug 
- sequelize 
- sqlite3
